# `psql` 客户端常用命令

## 1. 连接数据库

```bash
psql -h ${hostname} -p ${port} -U ${db_name}
```

## 2. 常用命令

### 2.1 列举数据库

`\l`：列出所有数据库的名字

`\l+`：列出所有数据库的名字以及字符集编码

### 2.2 切换数据库

`\c 库名`

### 2.3 列出库中的所有表

`\d`：列出当前库的所有表、索引、序列、视图、系统表、储存过程、视图

`\d{t|i|s|v|S|f|v}`：( 加 "+" 获取更多信息)列出:

+ `t`：表
+ `i`：索引
+ `s`：序列
+ `v`：视图
+ `S`：系统表
+ `f`：储存过程
+ `v`：视图

例如：

+ `\dt`：列出当前库的所有表

### 4、查看表结构

`\d table_name`

### 5、显示所有用户

`\du`

### 6、显示字符集

`\encoding`

### 7、退出 `psql`

`\q`
