# Notes

Personal notes

Powered by [vitepress](https://vitepress.vuejs.org/).

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Development

vitepress config: `docs/.vitepress/config.mts`

```bash
pnpm install
```

### 1) Build

Manually build the site:

```bash
pnpm docs:build
```

### 2) Preview

```bash
pnpm docs:preview
```
